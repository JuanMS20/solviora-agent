## Key Features

- **A closed learning loop** — Agent-curated memory with periodic persistence nudges, autonomous skill creation, skill self-improvement during use, FTS5 cross-session recall with LLM summarization
- **Runs anywhere, not just your laptop** — 6 terminal backends: local, Docker, SSH, Daytona, Singularity, Modal. Serverless environments hibernate when idle, costing nearly nothing
- **Lives where you do** — CLI, Telegram, Discord, Slack, WhatsApp, Signal, Matrix, Mattermost, Email, SMS — 15+ platforms from one gateway
- **Provider-agnostic** — OpenAI, Anthropic, Google Gemini, DeepSeek, Ollama, OpenRouter, or any OpenAI-compatible endpoint. Switch with a single config value
- **Scheduled automations** — Built-in cron with delivery to any platform
- **Delegates & parallelizes** — Spawn isolated subagents for parallel workstreams. Programmatic Tool Calling collapses multi-step pipelines into single inference calls
- **Open standard skills** — Compatible with agentskills.io. Skills are portable, shareable, and community-contributed
- **Full web control** — Search, extract, browse, vision, image generation, TTS
- **MCP support** — Connect to any MCP server for extended tool capabilities
- **Plugin system** — Custom tools, lifecycle hooks, and memory providers via a pluggable architecture

## Getting Started

```bash
# Install
pip install solviora-agent

# Set your API key
export OPENAI_API_KEY=sk-...

# Start chatting
solviora
```

## Why Solviora?

Solviora Agent is not a coding copilot tethered to an IDE or a chatbot wrapper around a single API. It is an autonomous agent that runs on your machine, connects to any LLM provider, and gets more useful the longer you use it.

**Three principles:**

1. **Your machine, your data** — everything stays local under `~/.solviora/`
2. **No lock-in** — switch providers with a single command
3. **Extensible by design** — tools, skills, plugins, and MCP for any workflow
