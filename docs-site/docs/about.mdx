# About Solviora Agent

## What is Solviora Agent?

Solviora Agent is a terminal-first, provider-agnostic AI agent for developers. It connects to multiple LLM providers through a unified interface, uses built-in tools to execute tasks, and runs entirely on your machine.

### Key principles

- **Terminal-native** — runs where developers work
- **Provider-agnostic** — no lock-in to any single LLM vendor
- **Extensible** — tools, skills, and plugins for every workflow
- **Privacy-first** — all data stays on your machine

## Origin

Solviora Agent is a fork of [Hermes Agent](https://github.com/NousResearch/hermes-agent) with its own identity and configuration paths. It maintains backward compatibility with `HERMES_HOME` for users migrating from Hermes.
