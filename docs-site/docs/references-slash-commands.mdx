# Slash Commands Reference

## Session Commands

| Command | Description |
|---------|-------------|
| `/help` | Show command help |
| `/clear` | Clear the current conversation |
| `/title <name>` | Name the current session |
| `/continue` | Resume the most recent session |
| `/resume <id>` | Resume a specific session by ID |

## Model Commands

| Command | Description |
|---------|-------------|
| `/model` | Show or change the current model |
| `/provider` | Show or switch the active provider |
| `/reasoning <level>` | Set reasoning effort (low, medium, high) |

## Tool Commands

| Command | Description |
|---------|-------------|
| `/tools` | List currently available tools |
| `/skills browse` | Browse the skills hub |
| `/skills list` | List installed skills |
| `/skills install <name>` | Install a skill |
| `/skills remove <name>` | Remove a skill |

## Session Context

| Command | Description |
|---------|-------------|
| `/memory` | Show memory status |
| `/memory search <q>` | Search past memories |
| `/compress` | Compress conversation context |
| `/usage` | Show token usage and cost breakdown |
| `/status` | Show session status |

## Voice Commands

| Command | Description |
|---------|-------------|
| `/voice on` | Enable voice mode |
| `/voice off` | Disable voice mode |
| `/voice tts` | Toggle spoken playback |

## Skin & Display

| Command | Description |
|---------|-------------|
| `/skin` | Show or switch the active CLI skin |
| `/skin <name>` | Switch to a specific skin |
| `/personality <name>` | Switch personality |

## Gateway Commands

| Command | Description |
|---------|-------------|
| `/gateway status` | Show gateway status |
| `/gateway restart` | Restart the gateway |
| `/gateway stop` | Stop the gateway |

## Workflow Commands

| Command | Description |
|---------|-------------|
| `/background <prompt>` | Run in a background session |
| `/queue` | Show queued background tasks |
| `/status` | Show queue and agent status |
