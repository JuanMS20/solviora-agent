# Proveedores

Solviora Agent es agnóstico al proveedor. Cambia de proveedor en cualquier momento con un solo ajuste de configuración.

## Proveedores soportados

| Proveedor | Config `provider` | Variable de entorno |
|-----------|-------------------|---------------------|
| OpenAI | `openai` | `OPENAI_API_KEY` |
| Anthropic | `anthropic` | `ANTHROPIC_API_KEY` |
| Google Gemini | `gemini` | `GOOGLE_API_KEY` |
| Mistral | `mistral` | `MISTRAL_API_KEY` |
| Groq | `groq` | `GROQ_API_KEY` |
| Together AI | `together` | `TOGETHER_API_KEY` |
| Fireworks AI | `fireworks` | `FIREWORKS_API_KEY` |
| DeepSeek | `deepseek` | `DEEPSEEK_API_KEY` |
| OpenRouter | `openrouter` | `OPENROUTER_API_KEY` |
| Cohere | `cohere` | `COHERE_API_KEY` |
| xAI | `xai` | `XAI_API_KEY` |
| AWS Bedrock | `bedrock` | Credenciales AWS |
| Azure OpenAI | `azure` | `AZURE_OPENAI_API_KEY` |
| Ollama | `ollama` | Ninguna (local) |
| LM Studio | `lmstudio` | Ninguna (local) |

## Configuración de ejemplo

### OpenAI

```yaml
provider: openai
model: gpt-4o
```

```bash
export OPENAI_API_KEY=sk-...
```

### Anthropic

```yaml
provider: anthropic
model: claude-sonnet-4-20250514
```

```bash
export ANTHROPIC_API_KEY=sk-ant-...
```

### Google Gemini

```yaml
provider: gemini
model: gemini-2.5-pro
```

```bash
export GOOGLE_API_KEY=...
```

## Proveedores locales

### Ollama

1. Instala [Ollama](https://ollama.ai)
2. Descarga un modelo: `ollama pull llama3.1`
3. Configura Solviora:

```yaml
provider: ollama
model: llama3.1
base_url: http://localhost:11434
```

### LM Studio

1. Instala [LM Studio](https://lmstudio.ai)
2. Carga un modelo y habilita el servidor local
3. Configura Solviora:

```yaml
provider: lmstudio
model: loaded-model-name
base_url: http://localhost:1234
```

## Cambiar de proveedor en tiempo de ejecución

```bash
solviora --provider anthropic --model claude-sonnet-4-20250514
```

O dentro de una sesión:

```
/model claude-sonnet-4-20250514
```

## Pool de credenciales

Si usas múltiples claves API para el mismo proveedor (por ejemplo, para rate limiting), Solviora soporta un pool de credenciales:

```yaml
credential_pool:
  openai:
    - key: sk-...
    - key: sk-...
```
