# Acerca de Solviora Agent

## Qué es

Solviora Agent es un agente de IA para terminal, diseñado para desarrolladores. Se conecta con múltiples proveedores de modelos de lenguaje (OpenAI, Anthropic, Google, etc.) y ejecuta herramientas de forma autónoma — lectura de archivos, ejecución de código, búsqueda web y más — dentro de una conversación interactiva.

## Origen

Solviora Agent es un fork de [Hermes Agent](https://github.com/NousResearch/hermes-agent), un proyecto de código abierto para agentes de IA en terminal. Solviora extiende la base con:

- Marca y configuración propias (`~/.solviora/`).
- Comandos CLI con nombre `solviora`.
- Documentación bilingüe (EN/ES).
- Ajustes y mejoras específicas del proyecto.

- Estándares de código y pruebas.

### Guía rápida

```bash
# Clonar
git clone https://github.com/JuanMS20/solviora-agent.git
cd solviora-agent

# Instalar en modo desarrollo
pip install -e ".[dev]"

# Ejecutar tests
scripts/run_tests.sh
```
