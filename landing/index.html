An agent infrastructure
for developers.

Solviora is an autonomous agent that runs on your own infrastructure — your server, a cloud VM, or a container. It connects to any LLM provider, persists what it learns across sessions, and extends through tools, skills, and MCP servers. Not a copilot. Not a chatbot wrapper. An agent you control.

1. Install
curl -fsSL https://raw.githubusercontent.com/JuanMS20/solviora-agent/main/scripts/install.sh | bash
2. Configure
solviora setup

Requires WSL2 on Windows.

solviora chat
$ solviora chat --model openai/gpt-4o
Using provider: openai · model: gpt-4o
Session started. Type /help for commands.

Capabilities

A focused set of primitives. Not a platform. Not an ecosystem. A tool that does what you ask.

Persistent memory

The agent stores what it learns across conversations. FTS5 recall, automatic summarization, and periodic persistence nudges. Optional pluggable backends.

Multi-platform gateway

CLI, Telegram, Discord, Slack, WhatsApp, Signal, Email — one gateway routes messages to the agent. Start on one, pick up on another.

Scheduled automations

Natural-language cron for reports, backups, and briefings. Runs through the gateway, delivers to any connected platform.

Subagent delegation

Spawn isolated agents for parallel workstreams. Each has its own conversation, terminal, and tools. No context-cost interference.

Sandboxed execution

Six terminal backends: local, Docker, SSH, Daytona, Singularity, Modal. Container hardening and namespace isolation.

Web and browser

Web search, browser automation with vision, image generation, text-to-speech. Works through any configured provider or dedicated API keys.

How it works

Three layers. The agent, the infrastructure it runs on, and the extensions it uses.

Agent core

  • Provider-agnostic LLM routing
  • Tool-calling loop
  • Memory and skill management
  • Subagent orchestration
  • Session persistence

Infrastructure

  • 6 terminal backends
  • Messaging gateway (15+ platforms)
  • Built-in cron scheduler
  • Plugin system
  • Profile isolation

Extensions

  • 68+ built-in tools (toolsets)
  • MCP server integration
  • Skill packages (agentskills.io)
  • Memory provider plugins
  • ACL + approval rules
